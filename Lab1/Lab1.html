Lab 1 - Artemis Nano



Objective

The goal of this lab was to get acquainted with the Artemis Nano board by running a few basic scripts on just the board with no periphrials.



Materials Used

  • 1 Artemis Nano
  • 1 USB A/C Cable
  • 1 Li-Ion 3.7V 500mAh battery


Procedure

To start with I downloaded the library for the board found, because I already had the Arduino IDE set up on my computer here
This link provides both instructions to set up the library and to run a few example programs
Once the board was set up, I ran three example programs:
1. The blink code to blink the built-in LED
2. The serial code to test the serial console 3. Code that observes and reports microphone frequency information
